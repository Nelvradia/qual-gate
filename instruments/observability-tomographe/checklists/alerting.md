# Alerting Checklist
- [ ] Process down alerts (core service, permission service, monitoring service)
- [ ] High memory usage (>80%)
- [ ] Disk space low (<10%)
- [ ] High error rate (>5% 5xx)
- [ ] Permission service unavailable
- [ ] Backup not running
- [ ] Certificate expiry (30d warning)
- [ ] Alert routing configured (email/push)
